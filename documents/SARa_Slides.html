cách học hiện tại

Lecture → quan sát bị động lâm sàng → về tự ôn 1 mình.

Phần 4 · Architecture

3 Agents song hành

cùng LLM, prompt khác
🩻

CV Agent

GPT-4o tiền xử lý prompt theo modality/region → MedGemma 4B (Gradio) trả findings JSON theo 4-step format.

💬

Socratic Agent

Đặt câu hỏi dẫn dắt theo bước hiện tại — không tiết lộ đáp án, tăng độ khó khi SV trả lời sai.

✅

Answer-Check Agent

Chấm câu trả lời theo rubric JSON đã được bác sĩ duyệt — chỉ ra sai ở bước nào, vì sao.

Phần 4 · Stack

Tech stack thực tế

đã chạy được
frontend
  • React 18 + Vite + TypeScript
  • Tailwind / module CSS
  • Volume-Slice viewer cho CT/MRI
  • Pages: Library · Session · Dashboard · AnswerKey
backend
  • Django + DRF
  • State machine 4 bước · score gate 0.6
  • Async job CV Agent (queue)
  • API: /sessions, /submit_answer, /cases
data & ai
  • Supabase Postgres · 12 bảng · RLS
  • Object Storage + Signed URL
  • MedGemma-1.5-4B qua gradio_client
  • GPT-4o pre-processing + rubric scoring
đã có

22 cases bulk-upload Supabase · multi-volume (Axial / Coronal / Bone Window) · answer key + rubric đã review · pipeline replay log.

tradeoff

VLM nhẹ → độ chính xác thấp hơn model nặng, nhưng đủ để luyện tư duy pipeline. Có fallback mock findings khi HF Space down.
